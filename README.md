# 🛍️ Virelle – E-commerce Website

## 📌 Overview

Virelle is a responsive e-commerce website built using HTML, CSS, and JavaScript. The project focuses on creating a smooth and visually appealing shopping experience with a clean UI and responsive layout across all devices.

## ✨ Features

* 📱 Fully responsive design (mobile, tablet, desktop)
* 🛒 Product listing and browsing
* 🔍 Product detail view
* 🧭 Easy navigation
* 🎨 Clean and modern user interface


## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript** – Interactivity

## 📂 Project Structure

```
projectvirelle/
│── index.html
│── css/
│   └── styles.css
│── js/
│   └── script.js
│── images/
```

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/jujjharshini17-arch/projectvirelle.git
```

### 2. Open the project

```bash
cd projectvirelle
```

### 3. Run the project

* Open `index.html` in your browser

## 🧪 Usage

* Browse products on the homepage
* Click on items to view more details
* Navigate using the menu
